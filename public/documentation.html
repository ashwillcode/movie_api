Movie API Documentation

This API allows users to search for and save information about movies. It includes endpoints for retrieving movie data, managing user accounts, and more.

Endpoints

Request URL Description
GET /movies Returns a list of all movies.
GET /movies/:title Returns data about a single movie by title.
GET /genres/:name Returns data about a genre by name.
GET /directors/:name Returns data about a director by name.
POST /users Registers a new user.
PUT /users/:username Updates user information.
POST /users/:username/movies/:movieId Adds a movie to a user's list of favorites.
DELETE /users/:username/movies/:movieId Removes a movie from a user's list of favorites.
DELETE /users/:username Deletes a user account.

GET /movies

Description: Returns a list of all movies.

Query Parameters: None

Body: Not applicable

Response:

      [
        {
          "title": "Inception",
          "director": "Christopher Nolan",
          "description": "A skilled thief, the absolute best in the dangerous art of extraction, stealing valuable secrets from deep within the subconscious during the dream state.",
          "genre": "Science Fiction",
          "featured": true,
          "imageURL": "https://m.media-amazon.com/images/I/51fKZ2s8XBL._AC_.jpg"
        },
        ...
      ]

GET /movies/:title

Description: Returns data about a single movie by title.

Query Parameters: :title (string) - Title of the movie (case-insensitive).

Body: Not applicable

Response:

      {
        "title": "Inception",
        "director": "Christopher Nolan",
        "description": "A skilled thief, the absolute best in the dangerous art of extraction, stealing valuable secrets from deep within the subconscious during the dream state.",
        "genre": "Science Fiction",
        "featured": true,
        "imageURL": "https://m.media-amazon.com/images/I/51fKZ2s8XBL._AC_.jpg"
      }

GET /genres/:name

Description: Returns all movies belonging to a specific genre.

Query Parameters: :name (string) - Name of the genre (case-insensitive).

Body: Not applicable

Response:

      [
        {
          "title": "Inception",
          "genre": "Science Fiction",
          ...
        }
      ]

GET /directors/:name

Description: Returns all movies directed by a specific director.

Query Parameters: :name (string) - Name of the director (case-insensitive).

Body: Not applicable

Response:

      {
        "director": "Christopher Nolan",
        "movies": ["Inception", "The Dark Knight", "Interstellar"]
      }

POST /users

Description: Registers a new user.

Query Parameters: None

Body: JSON object:

      {
        "username": "JohnDoe",
        "email": "john@example.com"
      }

Response:

      {
        "username": "JohnDoe",
        "email": "john@example.com",
        "id": 1
      }

PUT /users/:username

Description: Updates the username of an existing user.

Query Parameters: :username (string) - Current username of the user.

Body: JSON object:

      {
        "username": "NewUsername"
      }

Response:

      {
        "username": "NewUsername",
        "email": "john@example.com",
        "id": 1
      }

POST /users/:username/favorites

Description: Adds a movie to the user’s favorites list.

Query Parameters: :username (string) - Username of the user.

Body: JSON object:

      {
        "title": "Inception"
      }

Response:

      "Movie "Inception" added to favorites."

DELETE /users/:username/favorites

Description: Removes a movie from the user’s favorites list.

Query Parameters: :username (string) - Username of the user.

Body: JSON object:

      {
        "title": "Inception"
      }

Response:

      "Movie "Inception" removed from favorites."

DELETE /users/:username

Description: Deletes a user account.

Query Parameters: :username (string) - Username of the user.

Body: Not applicable

Response:

      "User has been deregistered."
